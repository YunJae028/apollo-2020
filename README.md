# apolloApp

React를 이용하여 간단한 최근 영화 웹을 만들었습니다.<br />
API로 데이터를 가져와서 Graphql을 통해서 구현했습니다.<br />

## Development Environment
- React
- Graphql
- Apollo
- node.js

## API
- YTS API (https://yts.mx/api)
